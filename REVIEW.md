# Профессиональный разбор сайта geragaraza.com

## Что сейчас хорошо (объективно)

1. **Минимализм** — дизайн чистый, ничего лишнего. Это правильный подход.
2. **CSS Custom Properties** — использование переменных `:root` — грамотно.
3. **`clamp()` для размеров** — адаптивная типографика без медиа-запросов. Профессиональный приём.
4. **Scramble-эффект** — добавляет характер. Хорошая деталь.
5. **Open Graph / Twitter Cards** — превью при шаринге ссылки. Грамотно.
6. **Системный шрифт SF Pro** — идеально для Apple-экосистемы.

## Что нужно исправить (критические моменты)

### 1. Accessibility (Доступность)
- `user-scalable=no` — **запрещает зум**. Это нарушение WCAG. Люди с плохим зрением не смогут увеличить страницу.
- Аватар-ссылка — нет `aria-label`, скринридер не поймёт, что это.
- Нет `:focus` стилей — при навигации с клавиатуры ничего не видно.
- Внешние ссылки без `rel="noopener noreferrer"` — потенциальная уязвимость.

### 2. SEO / Инфраструктура
- Нет `<link rel="canonical">` — дубликаты страниц в поисковиках.
- Нет `robots.txt` — поисковые боты не получают инструкций.
- Нет `sitemap.xml` — для одностраничника не критично, но профессионально.
- Нет `.nojekyll` — GitHub Pages может неправильно обрабатывать файлы.
- Нет `404.html` — при неверном URL пользователь увидит дефолтную страницу GitHub.
- README.md пустой — любой разработчик на GitHub увидит пустой репозиторий.

### 3. Код (HTML/CSS)
- Аватар — это `<a>` с фоном, а не `<img>`. Семантически неверно.
- `!important` в `.static-link` — плохая практика, признак борьбы с собственным кодом.
- Нет `prefers-reduced-motion` — люди, которым анимации мешают, не могут их отключить.
- Нет `prefers-color-scheme` — хотя сайт тёмный, это вопрос полноты.
- Hover-цвета в CSS без `@media (hover: hover)` — на тач-устройствах hover залипает.

### 4. JavaScript
- Scramble-эффект работает, но не очищает текст при быстром переключении вкладок.
- Нет `requestAnimationFrame` — `setInterval` менее производителен.
- Символы скрэмбла только латиница и цифры — можно сделать интереснее.

### 5. Производительность
- Нет `preconnect` для внешних ресурсов.
- Нет `dns-prefetch` для доменов соцсетей.
- Нет мета-тега `color-scheme: dark` — браузер может мигнуть белым при загрузке.

## Дизайн-философия новой версии

**Apple Way** — это не просто «минимализм». Это:
- Каждый пиксель имеет смысл
- Анимации плавные, с правильным easing (cubic-bezier)
- Типографика — главный герой
- Пустое пространство — это тоже дизайн
- Микро-взаимодействия дают ощущение «живого» интерфейса
- Ничего не мигает, не прыгает, не раздражает
